# Product Brief

## Working Idea

A lightweight software instrument that replaces the practical role of PC-98
soundfonts while giving the user a playable, editable synth instead of a static
sample bank.

The instrument should feel like a modern DAW plug-in, but its sound source
should be built from constrained chip-like behaviors:

- Game Boy-style drawable wave memory.
- Drawn-waveform resolution control.
- Low-resolution square/noise wave shapes, wave RAM editing, and table-driven
  bass, lead, pluck, and texture roles.

## What It Is

- A drawable stepped wavetable synth.
- A retro computer music color instrument.
- A retro wave-channel performance instrument for composers.
- A modern performance instrument with presets, envelopes, velocity, modulation,
  and DAW automation.

## What It Is Not

- Not an exact Game Boy APU emulator.
- Not an exact PC-98 sound-chip emulator.
- Not a ROM/sample dump or recreation of commercial PC-98 game assets.
- Not a register-level chip editor.

## Product Promise

The user can quickly create and play sounds built from low-resolution wave
memory, drawn-wave editing, and wave-table motion.

## MVP User

A composer or producer who wants:

- Chip-style wave-channel color without hunting for questionable soundfonts.
- Editable retro tones inside a modern DAW.
- Fast presets first, deep chip-style editing second.
